Commands

Arguments.

Required: <>

Optional: []

/shards

- get shards information

Usage:

/shards [date]

Provides detailed information about shattering shards in Sky: Children of the Light. Specify a date to get shard details for that day (e.g., "2023-09-15").

This command will give you insights into shard locations, landing times, and essential data for the selected date.

/sky-times

- get countdowns for various in-game times

Usage:

/sky-times [times]

Provides times for various in-game events and resets in Sky: Children of the Light.
This command offers a quick overview of the following times:
● Geyser time
● Grandma time
● Turtle time
● Reset time
● Eden reset time
● Events (Days of ...)

/shards-live

- configure live shards update

Usage:

/shards-live start <channel>: starts live shards update in the given channel

/shards-live stop: stops live shards update

Configure the channel where live shards update will be shown (updates every 5 minute)

/sky-times-live

- configure live sky-times updates

Usage:

/sky-times-live start <channel>: starts live shards update in the given channel

/sky-times-live stop: stops live sky-times update

Configure the channel where live sky-times update will be shown (updates every 2 minute)

Note: It is advisable to choose a channel for "-live" commands where no-one can send a message (except for the bot) so the bot's message doesn't get lost as the bot will be updating the same message sent during configuring the channel with live updates

/seasonal-guides

-get various seasonal seasonal guides

Usage:

/seasonal-guides [ephemeral]

Provides guides for seasonal quests, locations of seasonal spirits, and their cosmetic price trees, created by fellow game players. Results are ephemeral by default, but you can make them non-ephemeral optionally. The select menus are deleted after 1 minute of inactivity to prevent channel clogging.

/next-shards

- get upcoming shards details

Usage:

/next-shards [type] [number]

This command offers details on the date and location of the upcoming red shards. Use the optional [--number] flag to adjust the number of results. Default [number] is 5, max value for server is 10 and for bot's DM is 15 (To prevent channel clogs and respect text limits respectively)

/timestamp

- convert time into UNIX

Usage:

/timestamp <time> [timezone] [date] [month] [year] [format]

Converts a user-provided time into Unix time, using the default timezone (America/Los_Angeles) unless a different timezone is specified.

● <time>: Specify the time you want to convert. Format: HH mm ss (e.g, "12 45 20").
● [timezone]: Specify a timezone to use for the conversion (default: America/Los_Angeles).
● [date]: Specify a specific date (e.g., "15").
● [month]: Specify a specific month (e.g., "09").
● [year]: Specify a specific year (e.g., "2023").
● [format]: Specify the desired output format (e.g., "long-date", "short-date", "long-time-and-date", etc.).

For the convenience of iOS users, a copy link is available, providing direct access to the raw results for effortless copying.

/util

- utility commands

Usage:

/util credits: to works included in the bot

/util suggestions: suggest a feature or a change

/util changelog: bot's changelog
