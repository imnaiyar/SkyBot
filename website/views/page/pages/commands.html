Introduction

SkyHelper is a discord bot made to give useful informations related to Sky: Children of the Light game.

About this Bot

This bot provides informations related to Shards, Seasonal Guides, In-game event times such as Grandma, Turtle, etc.

Technology

The bot is built with nodejs using the discord.js library.

You can check bot's source codehere.

Getting Started

To get started with our bot is fairly easy and we have done our best to make it simpler for you.

In-bot help

You have the option to get help on all the bot commands right through discord with the use of the bot's internal /help command.

Important: You must agree to our Terms of Service and matter regarding it will be fairly strict.

Adding the bot

To add the bot to your server click here.

This does needs you to either be the owner of the server or have "manage server" permissions.

Commands

Syntax:
  • <> : Required Argument
  • [ ] : Optional Argument

Slash Commands/

  • Shards
    • Description : Let's you search Shards information. It's type, location and time of it's landing.
    Usage : /shards [date]
    Example : /shards
    /shards date:2023-09-16
  • Seasonal Guides
    • Description : Let's you search among various Seasonal Guides created by amazing Sky player. This includes, Seasonal Quests, Seasonal Spirits Location and Seasonal Spirits friendship tree(TS or Seasonal Price chart).
    Usage : /seseasonal-guides [ephemeral]
    Example : /seasonal-guides
    /seasonal-guides ephemeral: False
    Note : Some guides may become outdated as the game evolves.
  • Sky-Times
    • Description : Let's you know the upcoming (or currently active) in-game events times such as Grandma, Geyser, Turtle, Daily reset, Eden reset times.
    Usage : /sky-times [times]
    Example : /sky-times
    /sky-times times:Grandma Time
    /sky-times times:Geyser Time
    Note : By default it gives all times if none is selected, for more information, choose a time in optional subcommand.
  • Timestamp
    • Description : Let's you convert time into UNIX timestamp.
    Usage : /timestamp < time > [timezone][date] [month][year][format]
    Example : /timestamp 16 47 22
    Note : By default it converts to California time as TGC (Game's Dev) is based from there, but user can optionally provide their own timezone should they wish.

Prefix Commands!

    Usage : !skytimes [times]

  • Credits

    Usage : !credits
    Example : !credits
  • Ping
    • Description: Check Bot's ping.
    Usage : !ping
    Example : !ping
  • SkyGPT
    • Description : SkyGPT is based on OpenAI's ChatGPT, it answers to questions regarding the game Sky: Children of the Light. It has funny and sarcastic personality, and is trained on some of Sky's data. As, ChatGPT's knowledge is limited upto 2021, it often provides wrong information. It's good to check with other players. It's more for fun than anything.
    Usage : !skygpt set #channelname (Sets up the bot to listen for user's message in provided channel.)
    !skygpt stop(Stops the AI response in any and all channel it has been set for a server.)
    Example : !skygpt set #skyai-fun
    !skygpt stop
    Note : Bot will listen to any and all messages sent to the channel specified during setup and reply to it with it's response. You can make the bot ignore a message by adding ? before a message.
